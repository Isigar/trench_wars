BotI18nKeyCoverageTest enumeration:

Pattern: grep the codebase for `bot\.errors\.\w+` and `admin\.discord_outbound_message\.\w+` references in:
- apps/web/app/Http/Controllers/BotApi/
- apps/web/app/Http/Middleware/ResolveBotActsAsUser.php
- apps/web/app/Filament/Resources/DiscordOutboundMessageResource.php

For each found key, assert `trans($key)` returns a non-identity value (i.e., not the key itself echoed back).

Implementation:

  $files = File::glob(base_path('app/Http/Controllers/BotApi/*.php'));
  $files = array_merge($files, [
      base_path('app/Http/Middleware/ResolveBotActsAsUser.php'),
      base_path('app/Filament/Resources/DiscordOutboundMessageResource.php'),
  ]);

  $found = [];
  foreach ($files as $file) {
      preg_match_all("/__\(['\"]([a-z_]+\.[a-z_.]+)['\"]\)/", file_get_contents($file), $matches);
      foreach ($matches[1] as $key) $found[$key] = true;
  }

  foreach (array_keys($found) as $key) {
      $value = trans($key);
      expect($value)->not->toBe($key, "Translation key {$key} is missing from lang/en/.");
  }

  it() blocks:
  - it('every bot.* key referenced in BotApi controllers resolves')
  - it('every admin.discord_outbound_message.* key referenced in Filament resource resolves')
  - it('bot.errors.acts_as_unknown resolves to a localized string') — quick sanity check
