Route block (additive — APPEND to routes/api.php, do not replace):
```php
Route::middleware(['rcon.signature', 'throttle:600,1'])
    ->prefix('internal')
    ->name('internal.')
    ->group(function () {
        Route::post('match/{match}/events', [MatchEventsController::class, 'store'])
            ->whereUuid('match')->name('match.events.store');
        Route::get('bookings/due', [BookingScheduleController::class, 'dueNow'])
            ->name('bookings.due');
        Route::get('match-servers/{server}/credentials', [MatchServerCredentialsController::class, 'show'])
            ->whereUuid('server')->name('match-servers.credentials');
    });
```

```php
        'events.*.crcon_action' => ['nullable','string','max:128'],
        'events.*.payload' => ['required','array'],
        'events.*.occurred_at' => ['required','date'],
    ];
}
```

MatchEventsController::store contract:
- 202 Accepted + body `{ batch_id: <uuid>, accepted_count: <int> }` on success.
- 404 if route binding fails (unknown match).
- 422 with FormRequest validation errors on bad input.
- 401 from middleware on signature failure.

MatchServerCredentialsController::show contract:
- 200 + `{ host: string, port_rcon: int, api_token: string }`.
- 404 if server unknown OR `is_active=false`.
- Never logs the api_token (logger redact list extended via app/Providers/LoggingServiceProvider — defer to plan 08-12).

BookingScheduleController::dueNow contract:
- 200 + array of BookingDueData rows from `MatchServerBooking::active()->dueWithin(now()->subMinutes(5), now()->addMinutes(5))->with('server')->get()`.

|-----------|----------|-----------|-------------|-----------------|

| T-08-06-02 | Information disclosure | api_token leaks via response body cached/logged | mitigate | Response is text/json with no-cache; controller uses raw return (not `dd`); plan 08-12 extends Laravel log redact list to mask `api_token` paths. |
