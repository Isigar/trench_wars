**3. [Rule 2 — PHPStan testability hardening] Closure helper for player→username resolution**

- **Found during:** Task 1 (PHPStan L8 first run).
- **Issue:** PHPStan inferred `MatchPlayerStat->player` as `Player|null` (BelongsTo on a NOT-NULL FK is logically non-null but not declared as such in the model phpdoc), so the nested `$s->player?->display_name ?? $s->player?->slug ?? 'unknown'` chain tripped TWO L8 warnings ("nullsafe.neverNull" on the redundant `?->`, "Cannot access property $slug on Player|null" on the bare access). Plus Player.slug is NOT NULL at the DB layer but PHPStan reads it from $fillable inference, not the migration.
- **Fix:** Extracted a static closure `$resolveUsername(MatchPlayerStat $s): string` that explicitly type-checks `$s->player === null` once, then accesses `$player->display_name ?? $player->slug` with PHPStan-narrow types. Mvp lines + mvps data both call through the closure — one helper, two consumers.
- **Files modified:** `apps/web/app/Support/DiscordOutboundPayloadBuilder.php`.
- **Commit:** Folded into Task 1 commit `6adbe90`.
- **Plan correctness:** Output payload shape preserved bit-for-bit; only the local type-narrowing strategy changed.

**4. [Rule 2 — PHPStan testability hardening] Carbon stringify path for factory occurred_at**

- **Found during:** Task 1 (PHPStan L8 first run).
- **Issue:** `MatchEvent::occurred_at` is cast to `datetime` (model $casts) but PHPStan reads the column type as `string|null` from $fillable inference. The factory definition() puts `now()` (a Carbon instance) into the attribute bag; on a not-yet-persisted make()'d instance, that raw Carbon value persists. The initial `instanceof DateTimeInterface` check was flagged "always-false" because PHPStan thinks the attribute is `string|null`.
- **Fix:** Stringify via `(string) $raw` then `Carbon::parse(...)`. Carbon's __toString emits ISO-compatible output that parse() accepts losslessly; on a string input the round-trip is also lossless. The path handles both `now()` (Carbon) AND string inputs from chained `->state(['occurred_at' => '2026-05-14T...'])` calls.
- **Files modified:** `apps/web/database/factories/MatchEventFactory.php`.
- **Commit:** Folded into Task 1 commit `6adbe90`.
- **Plan correctness:** Wire output shape preserved bit-for-bit.
