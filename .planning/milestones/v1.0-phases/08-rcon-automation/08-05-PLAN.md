Wire format (every worker→web POST):
```
X-Rcon-Timestamp: <unix milliseconds, ASCII decimal>
X-Rcon-Nonce:     <UUIDv4 from worker>
X-Rcon-Signature: <lowercase hex HMAC-SHA256>
Content-Type:     application/json
```

Signing input: `timestamp_string + raw_body_bytes`. NOT `(method + path + ...)` — locked at CON-arch-rcon-to-web-comm to `(timestamp + body)` only.

Verification middleware (canonical PHP shape from RESEARCH lines 385-417):
```php
final class VerifyRconSignature
{
    public function __construct(private HmacVerifier $verifier) {}

    public function handle(Request $request, Closure $next): Response
    {
        $timestamp = $request->header('X-Rcon-Timestamp');
        $nonce     = $request->header('X-Rcon-Nonce');
        $sig       = $request->header('X-Rcon-Signature');
        $body      = $request->getContent(); // raw bytes — DO NOT $request->json()

        if (!$timestamp || !$nonce || !$sig) {
            abort(401, 'missing rcon auth headers');
        }
        $age = abs((int)(microtime(true) * 1000) - (int)$timestamp);
        if ($age > config('rcon.freshness_window_ms', 60_000)) {
            abort(401, 'stale signature');
        }
        if (!$this->verifier->verify((string)$timestamp, $body, $sig, config('rcon.hmac_secret'))) {
            abort(401, 'bad signature');
        }
        $stored = Redis::set("rcon:nonce:{$nonce}", '1', 'EX', config('rcon.nonce_ttl_seconds', 120), 'NX');
        if ($stored === null || $stored === false) {
            abort(401, 'replayed nonce');
        }
        return $next($request);
    }
}
```

HmacVerifier extracted as a separate class (testable in isolation, reusable in plan 08-09 TestConnection if needed):
```php
final class HmacVerifier
{
    public function sign(string $timestamp, string $body, string $secret): string {
        return hash_hmac('sha256', $timestamp . $body, $secret);
    }
    public function verify(string $timestamp, string $body, string $providedSig, string $secret): bool {
        $expected = $this->sign($timestamp, $body, $secret);
        return hash_equals($expected, $providedSig); // constant-time compare
    }
}
```
