TournamentSeedingService scaffold (Phase 4 final-class stateless idiom):

```php
<?php
declare(strict_types=1);
namespace App\Services;

use App\Exceptions\SeedingNotAllowedException;
use App\Models\Tournament;
use App\Models\TournamentParticipant;
use App\Models\User;
use Illuminate\Support\Facades\DB;

final class TournamentSeedingService
{
    public function __construct(
        private readonly TournamentStatusService $statusService,
    ) {}

    public function seed(Tournament $tournament, string $strategy, ?User $causer = null): void
    {
        DB::transaction(function () use ($tournament, $strategy, $causer): void {
            $participants = $tournament->participants()
                ->where('status', 'registered')
                ->orderBy('created_at')
                ->lockForUpdate()
                ->get();

            $ordered = match ($strategy) {
                'by_rank' => $this->orderByRank($participants),
                'random' => $participants->shuffle(),
                'manual' => $participants, // seeds already set by admin; just flip status
            };

            $count = 0;
            foreach ($ordered as $i => $participant) {
                if ($strategy !== 'manual') {
                    $participant->update([
                        'seed' => $i + 1,
                        'status' => 'active',
                    ]);
                } else {
                    $participant->update(['status' => 'active']);
                }
                $count++;
            }

            activity()
                ->causedBy($causer ?? auth()->user())
                ->performedOn($tournament)
                ->withProperties(['strategy' => $strategy, 'participant_count' => $count])
                ->log('Tournament seeded');
        });
    }

    public function reseed(Tournament $tournament, string $strategy, ?User $causer = null): void
    {
        if (! $tournament->canReseed()) {
            throw new SeedingNotAllowedException(__('tournaments.errors.reseed_not_allowed'));
        }

        DB::transaction(function () use ($tournament, $strategy, $causer): void {
            // Capture previous seeds (by clan_id for stable identity across re-seed)
            $previousSeeds = $tournament->participants()
                ->whereNotNull('seed')
                ->get()
                ->mapWithKeys(fn ($p) => [$p->clan_id => $p->seed])
                ->toArray();

            // Status: seeded -> registering (back-transition via TournamentStatusService)
            $this->statusService->transition($tournament, 'registering', $causer);

            // Reset participants back to 'registered' so seed() picks them up
            $tournament->participants()
                ->whereIn('status', ['active'])
                ->update(['status' => 'registered', 'seed' => null]);

            // Re-seed (this transitions participants back to 'active' and assigns seeds)
            $this->seed($tournament, $strategy, $causer);

            // Status: registering -> seeded (re-forward transition)
            $this->statusService->transition($tournament->refresh(), 'seeded', $causer);

            $newSeeds = $tournament->participants()
                ->whereNotNull('seed')
                ->get()
                ->mapWithKeys(fn ($p) => [$p->clan_id => $p->seed])
                ->toArray();

            activity()
                ->causedBy($causer ?? auth()->user())
                ->performedOn($tournament)
                ->withProperties([
                    'strategy' => $strategy,
                    'previous_seeds' => $previousSeeds,
                    'new_seeds' => $newSeeds,
                ])
                ->log('Tournament reseeded');
        });
    }

    /**
     * v1 by_rank: order participants by created_at desc as deterministic proxy for skill rank.
     * RESEARCH Assumption A11 — future Phase 9 polish swaps this for ELO-based ranking.
     *
     * @param \Illuminate\Database\Eloquent\Collection<int, TournamentParticipant> $participants
     * @return \Illuminate\Support\Collection<int, TournamentParticipant>
     */
    private function orderByRank(\Illuminate\Database\Eloquent\Collection $participants): \Illuminate\Support\Collection
    {
        return $participants->sortByDesc('created_at')->values();
    }
}
```

Tournament::canReseed() amendment (plan 06-03 model):

```php
// Add this method to apps/web/app/Models/Tournament.php
public function canReseed(): bool
{
    if ($this->status !== 'seeded') {
        return false;
    }

    // No MatchResult rows for any bracket-linked match in this tournament
    $matchIdsWithResults = \App\Models\MatchResult::query()
        ->whereIn('match_id', function ($q) {
            $q->from('tournament_brackets')
                ->select('match_id')
                ->whereNotNull('match_id')
                ->whereIn('tournament_stage_id', function ($q2) {
                    $q2->from('tournament_stages')
                        ->select('id')
                        ->where('tournament_id', $this->id);
                });
        })
        ->exists();

    return ! $matchIdsWithResults;
}
```

Performance note: the subquery uses 2 nested IN clauses against existing indexes (tournament_stages.tournament_id is indexed; tournament_brackets.tournament_stage_id is indexed; tournament_brackets.match_id is indexed). For tournaments with <= 64 participants × ceil(log2(64))=6 rounds = ~63 brackets, the query is O(63 + |MatchResult|) — well under the slow-query threshold. Phase 9 polish may add a materialised view if it surfaces.
