### PublicTournamentData::fromModel — nodes + edges composition

Single linear pass over `stages.brackets`. One `BracketNodeData` per bracket. For each bracket:

- if `advances_to_bracket_id !== null` → emit `BracketEdgeData(type='winner', to_slot=position%2===1 ? 'a' : 'b')`
- if `loser_advances_to_bracket_id !== null` → emit `BracketEdgeData(type='loser', to_slot=position%2===1 ? 'a' : 'b')`

8-clan single-elim canonical case: 7 nodes + 6 winner edges + 0 loser edges (the final has no `advances_to`, single-elim has no loser drop chain). 4-clan single-elim: 3 nodes + 2 winner edges with positions 1+2 filling slots a+b respectively.

### Etag shape (D-06-10-B)

```
etag = sha1(tournament.updated_at | sorted("$bracketId:$bracketUpdatedAt",...))
```

Deterministic across identical-state calls (tested for cross-call equality). Sort happens AFTER stringification on the `id:timestamp` token; bracket UUIDs are lexicographic. The JSON polling endpoint (plan 06-12) can short-circuit unchanged responses with HTTP 304 + If-None-Match.

### BracketNodeData 4-state status ladder

```
if (participantA && !participantB && winner) → 'bye'           // checked FIRST (D-06-10-C)
elseif (winner)                              → 'completed'
elseif (match_id)                            → 'in-progress'
else                                         → 'pending'
```

Bye is checked first because single-elim generators auto-set `winner_participant_id = participantA` for byes without materialising a `match_id`; the naïve `if winner → completed` ordering would mis-classify byes.

### TournamentObserver real impl (replaces plan 06-03 stub)

// saved() — polymorphic Event mirror (Phase 4 MatchObserver mirror)
$shouldHaveEvent = $tournament->is_public && $tournament->status !== 'cancelled';
if ($shouldHaveEvent) {
    Event::updateOrCreate(
        ['eventable_type' => Tournament::class, 'eventable_id' => $tournament->id],
        [
            'starts_at' => $tournament->starts_at ?? now(),  // D-06-10-G fallback
            'ends_at' => $tournament->ends_at,

    ->where('eventable_id', $tournament->id)
    ->delete();

// created() — tournament_announce outbound (Phase 5 plan 05-05 idiom)
if (! $tournament->is_public) return;
