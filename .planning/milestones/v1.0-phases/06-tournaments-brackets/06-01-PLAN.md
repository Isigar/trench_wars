Canonical Phase 4 factory stub idiom (verified `apps/web/database/factories/GameMatchFactory.php` from Phase 4 plan 04-01 commit 1d4d736):

- Namespace `Database\Factories;`
- `use Illuminate\Database\Eloquent\Factories\Factory;`
- `final class {Model}Factory extends Factory`
- `protected $model = 'App\\Models\\{Model}';` — STRING FQN not `::class` (model class does not exist yet)
- `/** @phpstan-ignore-next-line missingType.generics */` on the class line (Factory has generic param `<TModel>`)
- `/** @phpstan-ignore-next-line property.defaultValue */` on the $model property (PHPStan complains about a non-typed default)
- `public function definition(): array { return []; }` — empty body for Wave 0; plan 06-03 fills in attributes
- Add `final` keyword (Phase 4 idiom)
- File MUST not import the model class — string FQN bypasses class-existence check at parse time
