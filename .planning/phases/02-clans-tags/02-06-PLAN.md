<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| User-submitted clan name to ClanSlugGenerator | Untrusted input → slug → URL segment; reserved words must be rejected |
| Lang files to template render | __()/t() output is HTML-escaped by Blade/Vue automatically; lang files are owned source code |
| Vue components to Inertia props | All component-level data is typed via TypeScript generics |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-06-01 | Tampering | Reserved-slug bypass | mitigate | ClanSlugGenerator queries `config('clan.reserved_slugs')` and rejects via `throw new ReservedSlugException` if the candidate is in the list; FormRequest at plan 02-09 re-validates |
| T-02-06-02 | Information Disclosure | i18n value leak via :param interpolation | accept | Trans values flow through Laravel's interpolation helper which HTML-escapes by default in Vue when used via `{{ t(...) }}`; v-html is forbidden |
| T-02-06-03 | Tampering | Modal focus trap bypass | mitigate | Use Reka UI's `DialogContent` which ships built-in focus trap; do NOT override `inert` or `aria-modal` (UI-SPEC A11y) |
| T-02-06-04 | Information Disclosure | Hardcoded EN strings in Vue components | mitigate | NoHardcodedStringsTest (Phase 1) auto-scans `resources/js/components/`; every visible string must go through t() |
</threat_model>

    ```
    public function generate(string $name): string
    {
        $base = Str::slug($name);
        if ($base === '') {
            throw new \InvalidArgumentException('Cannot derive slug from name (empty after slug).');
        }
        if ($this->isReserved($base)) {
            throw new ReservedSlugException("Slug '{$base}' is reserved.");
        }
        $slug = $base;
        $i = 2;
        while (Clan::where('slug', $slug)->exists()) {
            $slug = $base . '-' . $i++;
        }
        return $slug;
    }
    ```
